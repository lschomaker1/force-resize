# force-resize
Forces windows that spawn off screen to resize to your display or a given amount of pixels - for Windows x86_64

**Usage:**

For example: 
- Open Powershell in the cloned directory where the executable is located or add to $PATH
- ```.\forceresize.exe "Bambu Studio" --popup```
- ```forceresize "Photoshop" 800 600 --popup```
